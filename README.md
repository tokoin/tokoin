## Install

```sh
git clone https://github.com/tokoin/tokoin.git
cd tokoin/
chmod +x scripts/build.sh
./scripts/build.sh
```

Now you can launch the application, by running the following command in the SAME directory:

```sh
./bin/tokoin
```
